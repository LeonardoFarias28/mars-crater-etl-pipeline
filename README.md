🚀 Mars Crater Analytics Pipeline (ETL + BI + ML)

This project goes beyond building a simple ETL pipeline. It is designed as a foundation for a complete data analysis and machine learning workflow.

📊 Phase 1: Data Analysis (Power BI)

After building the data pipeline, the next step is to explore and analyze the dataset using Power BI. The main objectives include:

Identifying the total number of craters
Analyzing crater size distribution
Exploring geographic patterns across Mars
🤖 Phase 2: Machine Learning

With a reliable and structured dataset, the project will evolve into advanced analytics, including:

Similarity analysis between craters
Pattern detection across regions
Clustering techniques to identify geological structures

🧠 Vision
The goal of this project is to demonstrate how a well-designed data pipeline enables scalable analytics and machine learning.
